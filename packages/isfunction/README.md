
# fnrepeats.isfunction

// NPM
npm i fnrepeats.isfunction@latest
// YARN
yarn add fnrepeats.isfunction@latest
```

```
const isFunction = require("fnrepeats.isfunction");

### Typescript
```
import isFunction from "fnrepeats.isfunction";
```
